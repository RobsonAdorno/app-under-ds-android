## [1.0.0] - 2026-03-22

### Added
- Tokens: UnderColors, UnderTypography, UnderSpacing, UnderShapes, UnderTheme
- Components: DSButton, DSTextField, DSCard, DSToolbar, DSDialog, FloatingNavBar
- Documentation for all tokens and components
- UX rules: error messages, visual feedback, status bar/insets
